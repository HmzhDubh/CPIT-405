PHP Basic Syntax

Variables: start with $ sign followed by the variable name which are case-sensitive, we can add line like js and statements ends with ;

Functions like js but it can take default parameter by giving the parameter a value thus it become default in case a parameter was not passed in the calling of the function

Arrays use regular brackets, and there are multiple types:

  • indexed(index keys): $arr = array("a", "b", "c")
  • Associative(string keys): $arr = array("a" => "hmzh", "b" => "dubh") in this case "a" takes the value "hmzh" and become more human-readable
  • Multidimensional: nested with different lengths

we can import PHP files using two statements

  1. include: file may be included but in cas in was not available to import the script will continue running, and it will produce a warning only
  2. required: file is required to complete the rest of the script, if the file is not available error will be produced and the script will stop
  3. Both of the above statements can be associated with _once, which means PHP will check if files already been included/required, if they are, then it won't get them again

PHP is often used to handle submitting forms

Validating form data on server side for security reasons, using "superglobals ($_GET and $_POST) to collect form data both of the contain array of data in name : value pairs

$_GET is used when data passed through URL parameters i.e. ?id=55 in the url and sent with the url to the server, which means data will be registered in the history so it should be to request data not submitting it specially if data sensitive like passwords

http://example.com/my_form.php?name=CPIT&number=405

$_POST -- via HTTP POST method, and it won't be registered in the history (more secure) and often used to submit data, where data sent in the bidy if the rquest not the url and the url is not added to the browser history
